Place rendered VHS demo GIFs here:

  deepstrain.gif  →  shown on deepstrain.dev hero
  atlas.gif       →  shown on deepstrain.dev/atlas

To render:
  cd /path/to/repo
  vhs demos/deepstrain.tape -o vercel-commerce/public/demos/deepstrain.gif
  vhs demos/atlas.tape      -o vercel-commerce/public/demos/atlas.gif

Install VHS:
  macOS:   brew install vhs
  Linux:   https://github.com/charmbracelet/vhs/releases
  Windows: winget install charmbracelet.vhs
